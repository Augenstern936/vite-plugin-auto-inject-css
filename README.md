<h1 align="center">vite-plugin-auto-inject-css</h1>

<p align="center">按需注入UI库对应的组件样式，无需手动引入</p>

## 📦 安装

```bash
$ npm i vite-plugin-auto-inject-css -D
```

```bash
$ yarn add vite-plugin-auto-inject-css -D
```

```bash
$ pnpm add vite-plugin-auto-inject-css -D
```
